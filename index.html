Development

Overview

Using React, I created a list interface with live filtering and sorting functionality for coffee pods.

Part 1: Architecture

Features to Include

I wanted my interface to include the following features:

  • 12 coffee pods, each displaying:
    • an image
    • name
    • cup size (espresso, double espresso, or coffee)
    • intensity
    • roast level (light, medium, or dark)
    • price
    • button to add to favorites
  • 2 filtering categories: roast level and cup size
  • 3 sorting features: alphabetically, by price, and by intensity
  • The total price of the favorited coffee pods

Functionality and Usability

Additionally, I wanted to include the following functions all while creating an intuitive design:

  • All filters and sorting methods work together without breaking the page
  • A method to revert back to the original state without refreshing the page
  • A method to favorite and unfavorite coffee pods (updating the total price accordingly)
